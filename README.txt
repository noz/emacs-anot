emacs-anot


* seamless editing
* per-file annotation files (*.anot)


Installation:
	(require 'anot)


Commands:

anot-insert
	insert annotation fragment at point
anot-region
	turn region into annotation
anot-un-annotation
	turn annotation into ordinary text

anot-toggle-show
	toggle show/hide annotations
anot-toggle-keep
	toggle keep-in/out annotations in the file
